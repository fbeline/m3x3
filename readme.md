# handwired/m3x3

My QMK 3x3 keyboard layout.

- **hardware**: atmega32u4
- **switch**: CHERRY MX BLUE
- **case**: [Scotto9](https://github.com/joe-scotto/scottokeebs/tree/main/Scotto9/Case)

**Keys range** from F13 to F21
